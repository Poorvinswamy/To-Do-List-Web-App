# 📝 To-Do List Web App

A simple and responsive **To-Do List Web Application** built using **HTML, CSS, and Vanilla JavaScript**. This project allows users to manage their daily tasks by adding, marking as complete, and deleting tasks without reloading the page.

---

## 📌 Project Description

The To-Do List Web App is a beginner-friendly front-end project that demonstrates the use of **DOM Manipulation**, **Event Handling**, and **JavaScript** to create an interactive task management application.

---

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- ⌨️ Add tasks by pressing the **Enter** key
- ⚡ Instant UI updates without page refresh
- 📱 Simple and responsive user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## ▶️ How to Run the Project

1. Download or clone this repository.
2. Open the project folder in **Visual Studio Code**.
3. Install the **Live Server** extension (optional).
4. Open `index.html` using Live Server or any web browser.
5. Start adding your tasks.

---

## 📸 Working

- Enter a task in the input field.
- Click the **Add** button or press **Enter**.
- Click on a task to mark it as completed.
- Click the **Delete** button to remove a task.
